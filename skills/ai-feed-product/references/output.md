# Output Reference

## Ranking Heuristics

Balance:

- freshness
- novelty
- credibility
- practical usefulness
- relevance to user interests

## Explanation Style

- prefer plain language
- give one concrete example
- explain expected user value
- include caveats where evidence is weak

## Trust Model

Separate:

- what the source explicitly says
- what surrounding evidence supports
- what the model infers as a likely use case
